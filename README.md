# ISO 27001:2022 主導稽核員模擬考試

線上模擬考試題庫 (ISO/IEC 27001:2022 Lead Auditor)。

## 本地預覽

直接用瀏覽器打開 `index.html` 即可。

## 部署

此專案為純靜態網站,可直接部署於:

- **Vercel** — 匯入 GitHub repo 即可 (無需設定 build command)
- **GitHub Pages** — Settings → Pages → Deploy from branch (main / root)
- **Cloudflare Pages / Netlify** — 連接 repo 即可
